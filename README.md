# Triton Gaming QR Studio

Single-purpose Next.js app for generating branded QR codes.

## Features

- Create QR codes from URLs or plain text
- Adjust canvas size and margin
- Customize background, module, and corner colors
- Pick module and corner styles
- Keep Triton Gaming logo presets or upload a custom icon
- Download the final QR code as a PNG

## Prerequisites

- Node.js 20+
- pnpm 10+

## Local development

```bash
pnpm install
pnpm dev
```

The app does not require any environment variables.

## Quality checks

```bash
pnpm lint
pnpm typecheck
pnpm build
```
